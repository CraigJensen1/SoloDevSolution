## What it does:

This is a game! When you first boot it up it will give you a menu in the CLI that you can use the keys on your keyboard to interact with. Up and down arrow keys to navigate the options, enter to actually select an option. From there you can choose to either start a new game, continue from where you last left off, or quit and leave. 

If you choose to create a new game you will be shown the most basic character creation of all time. You will have the ability to choose your character's name and gender. The game will actually have dialogue that takes these choices into account. Similar navigation as the main menu. Once that's done and you've confirmed your selection you'll be given some brief exposition before you can finally explore the grand open world.

In this open world you will have a variety of options, such as investigate irregular objects to discover secrets, defeat monsters to find cool new items, and interact with other people to progress the story. Once you get bored of all that or need to take a break, you have the ability to pause the game, save and quit. And once you boot the game back up you will have the option to pick up from where you last left off.

## Core features:

1. It has a menu that you can navigate through in order to access all of the other core features such as:

2. Starting a new game, which will make you a new character for your to play as which you can use to explore.

3. You have the power to pause the game and save it midway through so you can quit and come back later, in case you need to take a break for a while

4. It has the ability to load a previously saved game so you can pick up right where you last left off.

5. Upon entering the game, there is a map that you can navigate around using your keyboard's arrow keys!
